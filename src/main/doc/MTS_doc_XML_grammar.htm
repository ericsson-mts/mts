1          XML Grammar

  • schemas\scenario.html: => describe the entire list of MTS operations you can use in the scenario file.

 

  • schemas\test.html: => for the test file describe the test/testcase/scenario levels.

 

  • schemas\master.html: => for the master file describe the master/test levels.

 

  • schemas\scenario_protocol.html: => describe the entire list of protocol operation you can use in the scenario file (PPP means a generic protocol; replace it by the protocol acronym)

 

  • schemas\parameter_operation.html: => for the <parameter> operation describe the list of all sub-operation you can use (organized in categories)

 

  • schemas\test_condition.html: => for the <test> operation describe the list of all conditions you can use (organized in categories)
